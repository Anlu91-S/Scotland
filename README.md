# Scotland Roadbook 5.2 — Image-Led Discovery

A cleaner, image-first rebuild of the Scotland road-trip planner.

## Highlights
- Entire discovery cards are clickable; no redundant Read more buttons.
- Region pages use consistent editorial carousels for walks, food, stays, campsites and experiences.
- Detail pages begin with a navigable image gallery and end with the map, booking and official links.
- Key Skye, Cairngorms and Stirling imagery has been corrected locally.
- Official website preview images are discovered on a best-effort basis and cached in the browser.
- Tripadvisor is used as a secondary discovery/reputation source; AllTrails remains the trail reference.
- The route uses open destination descriptions rather than prescribing activities.

Upload the root files over the existing GitHub Pages files and replace/add the files inside `images/`.
