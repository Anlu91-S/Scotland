# Deployment checklist

1. Upload all root files and folders to GitHub.
2. Confirm `images/`, `worker/` and `tests/` remain folders.
3. Settings → Pages → Deploy from branch → `main` → `/(root)`.
4. Wait for the Pages deployment to turn green.
5. Open `https://anlu91-s.github.io/Scotland/?v=501`.
6. Optional: deploy `worker/`, add secrets and paste its URL into the app settings.
