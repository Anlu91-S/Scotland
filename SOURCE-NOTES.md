# Source notes

The roadbook prioritises official operator and destination websites for descriptions and image previews. Tripadvisor is used as a secondary discovery signal for restaurants, accommodation and attractions. AllTrails is used as the trail reference. Maps and navigation remain at the end of each detail page.

Key locally packaged corrections in this build include Quiraing, Fairy Pools, Old Man of Storr, Cairngorms reindeer and a more dramatic Stirling image. Dynamic official-site preview discovery is best-effort and gracefully falls back to locally packaged regional imagery if a website blocks preview extraction.
